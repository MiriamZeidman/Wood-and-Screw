# Garden Woodworks Website

## Project Description

A modern and professional website for a business specializing in pergolas, wooden decks, and custom garden woodwork.

The main goal of the website is to present a large and organized catalog of projects, divided into categories such as:

* Pergolas
* Wooden Decks
* Wooden Fences
* Custom Woodwork

Each project will include high-quality images and a short description.

In addition, the website will include a contact section where customers can leave their details, easily get in touch, and receive a quick response from the business.

The goal is to create a visual, professional, and user-friendly website that showcases the company’s work in an impressive way and helps attract new customers.
